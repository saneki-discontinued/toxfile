NAME

toxdump - dump tox files to other formats

SYNOPSIS

toxdump [options...] file

DESCRIPTION

toxdump is a utility for dumping tox files to other formats.

OPTIONS

-j, --json

Use JSON as the format to dump to.

-p, --profile=profile

Specify the profile name to use when looking for a tox file. This will look for ~/.config/tox/profile.tox and use it as the tox file if found.

-x, --include-private-key

Include the private key in the output.

-X, --hex-uppercase

Dump hex strings as uppercase. This affects the address, public key and private key.
